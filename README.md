# Equity-return-forecasting

Testing whether the next-day stock direction can be predicted from historial price data using supervised learning. covers six stocks across energy, tech and pharma (Aker BP, Equinor, Norsk Hydro, NVIDIA, Palantir, Novo Nordisk)

Main finding: simple models, like logistic regression, Naive Bayes, Decision Tree) do not beat a naive baseline, with AUC around 0,50, consistent with a weak-form market efficiency
